Questa è la quinta versione del mio sito web, progettata durante le giornate più calde di luglio 2023, otto anni dopo la precedente revisione, datata luglio 2015.

Ok, sono d’accordo, un professore di ingegneria web dovrebbe aggiornare il proprio sito un po’ più frequentemente. Nel frattempo molte cose sono cambiate… ad esempio, ho spostato tutte le pagine web e il materiale didattico in specifiche organizations di GitHub GitHub, quindi questo sito ora è online principalmente come riferimento per la mia attività di ricerca.

Per costruire un sito web moderno, facile da aggiornare, senza problemi di sicurezza, ho optato per un sito web statico realizzato con Hugo, con un design semplice, ampio e ottimizzato per i dispositivi mobili, creato da zero utilizzando Bootstrap e alcune immagini gratuite tratte da Freepik.

Ovviamente, il design è fluido e responsive, come dovrebbe essere per ogni sito web moderno. Gli script (tutti basati su JQuery) sono utilizzati per migliorare l’esperienza utente, ma le pagine sono pienamente fruibili anche con javascript disabilitato.

Buona navigazione!
